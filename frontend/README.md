# 🌸 Émo

Interface frontend de l'IA **Émo**. Pour l'utiliser, tu dois d'abord lancer le backend via Google Colab (voir le repo GitHub principal).

## Utilisation

1. Lance le notebook `colab/Emo_Backend.ipynb` sur Google Colab (GPU T4)
2. Copie l'URL ngrok affichée dans Colab
3. Colle-la dans le champ **API URL** de l'interface
4. Clique **Connecter** et discute !
